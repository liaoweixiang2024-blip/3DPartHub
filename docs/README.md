# 3DPartHub Docs

这个目录只保留通用项目文档。

- [运行环境规范](./运行环境规范.md)：本地开发、Docker 本地测试、生产部署的启动方式和端口约定。
- [并发压测与部署调优](./并发压测与部署调优.md)：浏览、静态资源、模型下载三类压力的压测命令和生产参数建议。

企业画册、产品批次、私有结构化数据、OCR 参考资料、Excel/PDF 原始资料和客户定制文档不放在 `docs/`。本地可临时放在仓库根目录的 `private-docs/`，该目录已被 Git 忽略；需要长期管理时请放到私有仓库或后台备份中。

历史批次导入脚本和私有种子数据也不要放在仓库根目录的 `data/`，本地统一放到 `private-docs/data/`。
